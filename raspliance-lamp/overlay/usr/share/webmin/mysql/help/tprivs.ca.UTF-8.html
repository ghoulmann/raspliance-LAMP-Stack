Permisos de Taula
La pàgina permet concedir permisos addicionals a les taules per part dels usuaris, més enllà del que està disponible a Permisos d'Usuaris o Permisos de Bases de Dades. Els permisos concedits aquí s'apliquen només a l'accés a una taula d'una base de dades per part d'un usuari, permetent així un accés molt restringit.

Per afegir nous permisos de taules, primer has de seleccionar la base de dades on està continguda la taula, utilitzant el formulari del peu de la pàgina. Després, fes clic sobre el botó Afegeix permisos nous per obrir un formulari per seleccionar la taula i altres detalls.
