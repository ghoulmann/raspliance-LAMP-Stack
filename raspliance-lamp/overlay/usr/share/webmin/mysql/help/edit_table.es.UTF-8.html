Editar Tabla
La tabla que llena la mayor parte de esta página muestra la estructura de la tabla seleccionada de tu base de datos. Cada fila detalla un campo dentro de la tabla, mostrando el nombre, tipo de datos y otras opciones.

En la parte inferior izquierda hay un botón para añadir un nuevo campo a la tabla. Antes de hacer click en él, debes de seleccionar el tipo de campo a añadir - mira la documentación de MySQL para más información sobre los tipos de campo. También en la parte inferior hay botones para ver todos los datos de la tabla y borrar la tabla entera y todos sus datos.
