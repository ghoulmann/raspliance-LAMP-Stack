Nombre de campo
El nombre de este campo en la tabla.

Valores enumerados (para tipos enum y set)
Una lista de valores posibles para este campo.

Ancho y decimales (para tipos float y decimal)
La medida total de este campo y el número de dígitos a la derecha del punto decimal.

Ancho del tipo (para otros tipos)
El número máximo de caracteres permitidos para los datos de este campo.

Opciones del tipo
Para campos numéricos, este parámetro te permite seleccionar si el campo es sin signo o con relleno de ceros a la izquierda. Para campos de carácter, puedes seleccionar si las mayúsculas se tienen en cuena en las clasificaciones o no.

¿Permitir nulos?
Determina si los valores NULL están autorizados en este campo

Valor por defecto
El valor por defecto para este campo en los nuevos registros
