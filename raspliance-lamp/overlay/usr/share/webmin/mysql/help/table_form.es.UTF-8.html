Crear Tabla
Este formulario es para crear nuevas tablas vacías. Debes de digitar un nombre de tabla y uno o más campos. Para cada campo, debes de digitar un nombre, seleccionar un tipo de dato y digitar el ancho del tipo (si es apropiado para el tipo de dato). Al terminar, haz click en el botón Crear para hacer que la tabla quede inmediátamente creada.
