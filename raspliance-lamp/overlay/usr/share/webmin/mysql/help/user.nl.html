Voor iedere gebruiker zijn de volgende parameters beschikbaar :
Gebruikersnaam
In dit veld kunt u een specifieke gebruikersnaam invullen, of de kies de Anonieme gebruiker optie zodat alle permissies worden toegevoegd aan iedere gebruiker die niet in de Gebruiker Permissies lijst staan.

Wachtwoord
Het wachtwoord wat nodig is voor deze gebruiker om in te loggen. Indien Niets, word ingevuld is er geen wachtwoord nodig om in te loggen.

Hosts
Het IP adres of de hostnaam van de host vanwaar deze gebruiker mag inloggen. Dit veld mag ook een SQL reguliere expressie bevatten die overeenkomt met meerdere hosts, zoals 192.168.1.% of %.foo.com.

Permissies
De MySQL operaties die deze gebruiker mag uitvoeren op iedere database.
