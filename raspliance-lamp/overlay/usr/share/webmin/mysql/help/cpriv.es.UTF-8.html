Los siguiente parámetros están disponibles:
Campo
El campo al que se deberían aplicar esto permisos.

Nombre de usuario
El usuario al que se condecen los permisos. Este usuario ya debe de existir en la tabla de Permisos de Usuario.

Máquinas
Las máquinas desde las que el usuario puede acceder a la tabla.

Permisos
Los permisos concedidos al usuario en la columna seleccionada de la tabla.
