Följande inställningar kan göras:
Tabell
Tabell som rättigheterna gäller för.
Användarnamn
Användare som får rättigheterna. Användaren måste finnas i tabellen Användarrättigheter
Datorer
Datorer som användaren kan komma åt tabellen från.
Tabellrättigheter
Användarens rättigheter för samtliga kolumner.
Fälträttigheter
Maximala rättigheter som denna användare kan få i någon tabellkolumn på sidan Fälträttigheter
