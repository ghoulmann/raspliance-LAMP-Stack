De volgende parameters zijn beschikbaar :
Databases
De databases waar toegang aan verleent is. Dit kan een SQL regexp zijn indien dit gewenst is.

Gebruikersnaam
De gebruiker die toegang heeft bij de database die hierboven staat. Deze gebruiker moet reeds bestaan in de Gebruikers Permissies tabel.
Hosts
De hosts vanwaar de gebruiker toegang heeft tot de database. Indien de optie Van host permissies is gekozen, zal MySQL de Host Permissies gebruiken om de toegang te controleren.

Permissies
De MySQL handelingen die de gebruiker mag uitvoeren op de database.
