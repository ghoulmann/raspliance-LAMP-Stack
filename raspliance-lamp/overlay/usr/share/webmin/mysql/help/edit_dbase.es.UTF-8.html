Editar Base de Datos
En la parte superior de esta página hay filas de iconos, cada uno representando una tabla dentro de la base de datos. Para editar la estructura de una tabla, sólo haz click en uno de los iconos.

Debajo hay botones para crear una nueva tabla dentro de esta base de datos, ejecutando SQL en las tablas de la base de datos y borrando la base de datos y todas las tablas de ella.

¡El botón final debería de usarse con cuidado ya que no es reversible!.
