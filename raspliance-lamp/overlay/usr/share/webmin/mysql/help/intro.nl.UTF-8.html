MySQL Database Server
MySQL is een simpele database server voor Unix systemen. Het ondersteund meerdere databases, heeft een krachtige toegangscontrole, ondersteund veel soorten data en de meeste standaard SQL syntaxis. Iedere database die beheerd word door de server kan meerdere tabellen bevatten, en iedere tabel kan meerdere rijen met data bevatten.

Op de hoofdpagina word een lijst getoond met databases en iconen aan de bovenkant, met een link eronder om een nieuwe database toe te voegen. Iedere MySQL server heeft op zijn minst de mysql database, waarin de configuratie en toegangscontrole tabellen in zijn opgeslagen. Onder de databases bevinden zich iconen om de toegangscontrole te configureren en helemaal onderaan bevind zich een knop om de database server te starten en te stoppen.
