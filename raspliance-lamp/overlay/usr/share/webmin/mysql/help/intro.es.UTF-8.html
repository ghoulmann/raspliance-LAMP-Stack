Servidor de Base de Datos MySQL
MySQL es un servidor de base de datos simple para sistemas Unix. Soporta múltiples bases de datos, potentes controles de acceso, muchos tipos de datos y la mayor parte de la sintáxis SQL estándar. Cada base de datos gestionada por el servidor puede contener múltiples tablas y cada tabla contener múltiples filas de datos.

La página principal muestra una lista de iconos de base de datos en la parte superior, con un enlace para añadir una nueva base de datos bajo cada uno de ellos. Cada servidor MySQL tendrá al menos la base de datos mysql que almacena tablas de configuración y de control de acceso. Bajo las bases de datos hay iconos para configurar el control de acceso y en la parte inferior hay un botón para parar y arrancar el servidor de base de datos.
