Följande inställningar kan göras:
Databaser
De databaser som ska vara tillgängliga. Om du vill kan du skriva in ett reguljärt SQL-uttryck.
Användarnamn
Den användare som får komma åt databaser(erna) ovan. Användaren måste finnas i tabellen Användarrättigheter.
Datorer
Datorer som användaren får komma åt databasen från. Om du har valt Rättigheter från dator kommer MySQL att använda Datorrättigheter för att kontrollera åtkomsten.
Rättigheter
De MySQL-operationer som användaren får utföra på databasen.
