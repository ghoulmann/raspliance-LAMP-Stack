Plazo para completar las conexiones TCP
smtp_connect_timeout
Este parámetro especifica el plazo en segundos durante el que se intenta completar una conexión TCP como cliente SMTP.

Si no se establece conexión alguna antes del plazo el cliente SMTP probará la siguiente dirección en la lista de intercambiadores de correo. Especifique 0 para desactivar el plazo máximo de espera.
