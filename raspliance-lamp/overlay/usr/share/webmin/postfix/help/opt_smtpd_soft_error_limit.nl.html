Fouten telling voor het tijdelijk weigeren van een cliënt
smtpd_soft_error_limit
Deze parameter specificeert een onder limiet voor de fouten telling. Wanneer een SMTP cliënt zijn aantal fouten heeft gemaakt in een sessie zal de server gaan wachten met error_count aantal seconden voordat hij reageert op een cliënt aanvraag.
