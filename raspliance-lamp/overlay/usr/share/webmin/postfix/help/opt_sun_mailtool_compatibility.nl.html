Uitschakelen van de kernel file slot op mailboxen
sun_mailtool_compatibility
Deze parameter schakelt de kernel file sloten uit op mailboxen.

Dit is nodig op SUN werkstations omdat het mailtool programma een exclusief kernel slot vasthoud zolang de window open staat. SUN software gebruikt alleen de user.lock files. Indien u niet alle SUN email software verwijdert zal de kernel sloten u alleen maar een vals gevoel van veiligheid geven.
