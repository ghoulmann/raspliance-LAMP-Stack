Shell om te gebruiken voor levering aan externe opdracht
local_command_shell
Deze parameter bepaald welke shell word gebruikt voor het leveren aan externe opdrachten. Standaard worden externe opdrachten direct uitgevoerd : de opdrachten worden alleen gegeven aan /bin/sh wanneer deze een shell meta karakter bevatten of ingebouwde shell opdrachten.

Specificeer een andere shell wanneer u meer controle wenst over wat voor soort programma's moeten worden uitgevoerd van bijvoorbeeld de .forward files.

"Sendmail zijn beperkte shell" smrsh is wat de meeste mensen zullen gebruiken. (smrsh is een onderdeel van de recente Sendmail distributies)

Let op : wanneer er een shell word gespecificeerd, word deze altijd aangeroepen, zelfs als de opdracht geen ingebouwde shell opdrachten of meta karakters bevat.

Voorbeeld : /some/where/smrsh -c
