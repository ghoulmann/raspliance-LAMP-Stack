Nivel inicial de cocurrencia para entrega al mismo destino
initial_destination_concurrency
Este parámetro especifica el nivel de cocurrencia inicial por-destino para entrega paralela al mismo destino. Este límite se aplica a entrega vía SMTP, vía agente de entrega local y vía cartero por tubo.

Con concurrencia de 1, un mensaje malo es suficiente para bloquear todo el correo a un sitio. Una concurrencia de 2 parece una elección razonable.
