Resposta del servidor SMTP en cas de rebuig per nom de host desconegut
unknown_hostname_reject_code
Aquest paràmetre especifica el codi de resposta del servidor SMTP quan un viola la restricció reject_unknown_hostname, això és: Accés denegat: Nom de host HELO del client desconegut (sense registre DNS A o MX).

Tingues en compte que el valor per defecte utilitzat per postfix per a unknown_hostname_reject_code és el codi 501, i aquest codi en l'estàndard RFC 822 voldria dir 'Error de sintaxi en els paràmetres o arguments'. Així, en Postfix, un codi d'estat 501 no reflecteix exactament l'explicació de RFC822.

Per evitar confusions pots canviar-ho, per exemple, pel 577 (un altre motiu és que el codi 501 també és el valor per defecte de postfix d'altres menes d'errors), altrament no ho canviïs si no tens un complet coneixement de RFC 822.
