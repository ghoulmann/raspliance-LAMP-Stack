Pad naam van de gebruiker mailbox file
home_mailbox
Deze parameter specificeert de optionele pad naam van een mailbox file, relatief ten opzichten van een gebruikers home directory. De standaard mailbox file is normaal /var/spool/mail/user of /var/mail/user.

Specificeer "Maildir/" voor qmail-stijl aflevering (de / is nodig).

Voorbeelden :

  • Mailbox
  • Maildir/
