Tidsgräns vid väntan på svar på avslutande "."
smtp_data_done_timeout
Denna parameter anger den tid (i sekunder) som SMTP-klienten har på sig för att skicka SMTP "." och för att ta emot serversvaret.

Om inget svar tas emot inom tidsgränsen loggas en varning om att e-postmeddelanden kan komma att skickas flera gånger.

Vid problem kommer klienten INTE att testa nästa adress på mail exchanger-listan.
