Dirección destino para destinatarios desconocidos
luser_relay
Este parámetro especifica una dirección opcional de destino para destinatarios desconocidos. Por defecto, el correo para destinatarios locales desconocidos es enviado.

Se realizan las siguientes expansiones:

  • $user (nombre de usuario de destinatario),
  • $shell (shell de destinatario),
  • $home (directorio inicial de destinatario),
  • $recipient (dirección completa de destinatario),
  • $extension (extensión de dirección de destinatario),
  • $domain (dominio de destinatario),
  • $local (parte local completa de destinatario),
  • $recipient_delimiter.
Especifica ${name?value} o ${name:value} para expandir el valor sólo cuando $name (no) exista.

Ejemplos:

  • $user@otra.máquina
  • $local@otra.máquina
  • admin+$local
