Note: dat de Postfix standaard waarde die gebruikt word voor unknown_address_reject_code code 450 is, echter die word ook voor andere fouten gebruikt. Deze code in 'standaard RFC 822' betekend 'Aangevraagde email actie niet uitgevoerd: mailbox in gebruik of toegang geweigerd' en vraag de afzender het later nogmaals te proberen (hopende dat de DNS dan wel correct reageert).
