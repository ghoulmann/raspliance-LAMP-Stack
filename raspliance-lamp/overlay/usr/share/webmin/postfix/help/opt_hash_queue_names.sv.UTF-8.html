Namn på kökataloger som delas upp på underkataloger
hash_queue_names
Denna parameter anger namnen på de kökataloger som delas upp på flera underkatalognivåer. Hashning får inte användas med en e-postkatalog som hela världen kan skriva till. Hashning måste användas för fördröjningsloggfilskatalogen, annars kommer e-postsystemets prestanda att gå ner.

Tyvärr kan hashning av kön för inkommande meddelanden eller fördröjningskön faktiskt slöa ned e-postsystemet (mailq på en tom kö kan ta flera sekunder), så detta bör endast göras i nödfall.
