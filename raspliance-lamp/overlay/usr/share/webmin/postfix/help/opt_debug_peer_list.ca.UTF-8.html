=Llista de patrons de dominis/xarxes pels quals està activat el registre detallat
debug_peer_list
Aquest paràmetre especifica una llista opcional de dominis o patrons de xarxa, patrons /nom/fitxer o taules tipus:nom. Quan un client SMTP, nom de host servidor o adreça coincideix amb un patró incrementa el nivell de verbositat dels registres amb el valor especificat pel paràmetre debug_peer_level

Possibles valors sensibles:

  • 127.0.0.1
  • algun.domini
