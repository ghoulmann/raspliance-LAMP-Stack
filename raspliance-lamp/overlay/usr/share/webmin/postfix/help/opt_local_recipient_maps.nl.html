Weiger onbekende lokale gebruikers
local_recipient_maps
Deze parameter specificeert optionele opzoek tabellen met alle gebruikers die lokaal bekend zijn in de $mydestination ("welke domeinen mail mogen ontvangen ") en $inet_interfaces ("netwerk interfaces voor het ontvangen van mail"). Indien deze parameter word gedefinieerd, zal de SMTP server mail weigeren van onbekende lokale gebruikers.

Gebruik tabellen met klare gebruikersnamen zoals unix:passwd.metnaam en alias mappen. Voorbeeld : $alias_maps unix:passwd.metnaam.

Let op : indien de Postfix SMTP server chrooted werkt, moet de wachtwoord database kopieren in de jail. Dit is wel systeem afhankelijk.
