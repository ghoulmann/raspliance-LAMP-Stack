Plazo en segundos para las transacciones SMTP
opts_smtpd_timeout
Este parámetro limita el tiempo en segundos para enviar una respuesta SMTP desde el servidor y para recibir una petición desde un cliente SMTP.

Nota: si pone valores altos a los plazos SMTP deberá actualizar también el parámetro ipc_timeout, que puede encontrar en la sección Opciones Generales.
