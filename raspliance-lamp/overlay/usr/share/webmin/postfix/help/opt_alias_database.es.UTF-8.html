Base de datos de alias construídas por Postfix
alias_database
Este parámetro especifica la base de datos de alias que son construídas con "newaliases" o "sendmail -bi". Este es un parámetro a parte de la configuración debido a que alias_maps puede especificar tablas que no están necesáriamente bajo el control de Postfix.

Las típicas configuraciones son:

  • dbm:/etc/aliases
  • dbm:/etc/mail/aliases
  • hash:/etc/aliases
  • hash:/etc/aliases, hash:/opt/majordomo/aliases
