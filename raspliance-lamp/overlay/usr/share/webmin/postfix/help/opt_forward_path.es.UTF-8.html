Lista de búsquda para remisión
forward_path
Este parámetro especifica una lista de búsqueda. Se usa el primer archivo que se encuentra.

Se realizan las siguientes expansiones:

  • $user (nombre de usuario de destinatario),
  • $shell (shell de destinatario),
  • $home (directorio inicial de destinatario),
  • $recipient (dirección completa de destinatario),
  • $extension (extensión de dirección de destinatario),
  • $domain (dominio de destinatario),
  • $local (parte local completa del destinatario),
  • $recipient_delimiter.
Especifica ${name?value} o ${name:value} para expandir valor sólo cuando $name (no) exista.
